# Router Interventions Notebooks

Jupyter notebooks for visualizing and analyzing router intervention experiments.

## Notebooks

### `project_out_plots.ipynb`
Visualizes project-out experiment results:
- **Delta vs k**: Loss change when projecting out top-k singular vectors (single layer)
- **Delta vs layer**: Loss change across layers (fixed k)

Load results from `results/` folder (generated by `run_project_out.py`).

### `router_intervention_plots.ipynb`
Bar charts for single-run ablations showing loss and delta by variant.

### `router_subspace_ablation_plots.ipynb`
Subspace ablation analysis (legacy, may need updating).

## Usage

```bash
# From repo root
jupyter notebook src/experiments/router_interventions/notebooks/
```

Or open directly in VS Code / Cursor.
